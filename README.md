# Dlib Wheels Builder

Build dlib wheels for multiple Python versions using GitHub Actions.

## How to Use

1. **Push to GitHub** - Workflows trigger automatically on push to `main`
2. **Or trigger manually** - Go to Actions → "Build dlib Wheels" → "Run workflow"
3. **Download wheels** - After build completes, download from Artifacts section

## Supported Python Versions

- Python 3.8
- Python 3.9
- Python 3.10
- Python 3.11
- Python 3.12
- Python 3.13

## Build Time

~15-20 minutes total (builds run in parallel)
